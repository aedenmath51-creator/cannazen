## 🎯 CONTEXTE : CE QUI EXISTE DÉJÀ

Tu travailles sur un projet **pnpm monorepo** entièrement structuré. La quasi-totalité du code est écrite. Voici ce que le projet contient :

### Architecture monorepo
```
/
├── artifacts/
│   ├── cannazen/          → Frontend React + Vite (port 8081)
│   └── api-server/        → Backend Express 5 (port 8080)
├── lib/
│   ├── db/                → Drizzle ORM + schémas PostgreSQL
│   ├── api-spec/          → OpenAPI YAML
│   ├── api-zod/           → Schémas Zod générés
│   └── api-client-react/  → Hooks React générés (React Query)
└── scripts/
```

### Stack technique exacte
- **Runtime** : Node.js 24, pnpm workspaces
- **Frontend** : React 18 + Vite + Wouter (routing) + Framer Motion + TanStack Query
- **UI** : Tailwind CSS v4 + shadcn/ui + Lucide React
- **Backend** : Express 5 + TypeScript
- **Base de données** : PostgreSQL 16 + Drizzle ORM
- **Validation** : Zod v4 + drizzle-zod
- **Build** : esbuild
