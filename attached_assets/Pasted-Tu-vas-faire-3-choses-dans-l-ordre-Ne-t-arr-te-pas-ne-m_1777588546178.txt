## MISSION 2 — CORRIGER LE SITE AU PIXEL PRÈS

Ouvre https://www.cannazen.best dans ton navigateur intégré et inspecte chaque page. Corrige tous ces problèmes dans le code :

### Espacement et typographie
- Vérifie que toutes les marges entre les sections (`margin-top`, `padding`) correspondent exactement au design de cannazen.best
- Vérifie les espacements entre les items de navigation dans le header (gaps entre onglets)
- Vérifie les espacements entre les titres et les paragraphes (`line-height`, `letter-spacing`)
- Vérifie que les boutons ont le bon `padding` intérieur (ni trop serrés, ni trop larges)
- Vérifie l'espacement des cards produits dans la grille (gap entre cards)

### Navigation / Header
- Les onglets du menu doivent être correctement espacés entre eux
- Le logo doit être bien centré ou à gauche selon le design original
- Le panier, le compte, la recherche doivent être alignés à droite sans chevauchement
- Sur mobile : le menu hamburger doit s'ouvrir correctement avec la bonne animation
- Ajoute un `position: sticky; top: 0; z-index: 50;` sur le header si pas déjà fait

### Footer
- Vérifie que les colonnes du footer sont bien alignées
- Les liens doivent avoir un `hover` visible
- L'espacement entre les sections du footer doit être cohérent

### Fiche produit
- L'image principale doit prendre exactement la bonne largeur
- Le prix doit être mis en valeur (taille, couleur, gras)
- Le bouton "Ajouter au panier" doit être pleine largeur sur mobile
- Les informations CBD% / THC% doivent être affichées proprement en badges

### Pages globales
- Ajoute `scroll-behavior: smooth` sur le `html`
- Vérifie que toutes les pages ont le même `max-width` et `margin: 0 auto` sur le container
- Vérifie que les titres `h1`, `h2`, `h3` ont une hiérarchie visuelle claire et cohérente
- Toutes les images doivent avoir `loading="lazy"` sauf celles above-the-fold
- Aucun texte ne doit déborder de son container sur mobile (vérifie `overflow-x: hidden` sur le body)

### Pop-up vérification d'âge
- Doit apparaître centré, avec un backdrop sombre
- Ne doit pas pouvoir être fermé en cliquant en dehors
- Le bouton "J'ai 18 ans" doit être bien visible
- Le bouton "Je n'ai pas 18 ans" doit exister et rediriger vers google.com

### Checkout
- Les 3 étapes doivent avoir une progress bar visible en haut
- Les champs de formulaire doivent avoir des labels et des placeholders clairs
- Les messages d'erreur de validation doivent apparaître sous chaque champ en rouge
- Le bouton "Payer" doit afficher un spinner pendant le traitement Mollie
